# Plan: 개인용 퀀트 트레이딩 플랫폼

## 1. 개요

| 항목 | 내용 |
|------|------|
| Feature | quant-trading-platform |
| 작성일 | 2026-03-06 |
| 상태 | Plan |
| 우선순위 | High |

### 1.1 목적
개인 투자자를 위한 퀀트 트레이딩 플랫폼을 구축한다. 한국/미국 주식 데이터를 수집하고, 다양한 트레이딩 전략을 백테스팅하며, 분석 결과를 시각화하여 데이터 기반 투자 의사결정을 지원한다.

### 1.2 범위
- 주식 데이터 수집 및 관리 (FinanceDataReader)
- 퀀트 전략 개발 프레임워크
- 백테스팅 엔진 (backtrader)
- 분석 결과 시각화 (matplotlib)

## 4. 핵심 기능 요구사항

### FR-01: 데이터 수집
- FinanceDataReader를 통한 한국/미국 주식 가격 데이터 수집
- 일별 OHLCV(시가, 고가, 저가, 종가, 거래량) 데이터 지원
- CSV 파일로 로컬 저장 및 캐싱

### FR-02: 전략 개발
- backtrader 기반 전략 클래스 템플릿 제공
- SMA 크로스오버, 모멘텀 등 기본 전략 포함
- 사용자 정의 전략 추가 용이한 구조

### FR-03: 백테스팅
- backtrader 엔진을 활용한 전략 백테스팅
- 수익률, 최대 낙폭(MDD), 샤프 비율 등 성과 지표 계산
- 기간별 백테스팅 지원

### FR-04: 시각화
- 주가 차트 및 기술적 지표 시각화
- 백테스팅 결과 시각화 (수익률 곡선, 드로다운 차트)
- 전략 비교 차트
