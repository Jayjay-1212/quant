# Plan: 전략 파라미터 최적화

## 1. 개요

| 항목 | 내용 |
|------|------|
| Feature | strategy-optimizer |
| 작성일 | 2026-03-06 |
| 상태 | Plan |
| 대상 파일 | backtest/optimizer.py |

### 1.1 목적
그리드 서치 방식으로 전략 파라미터의 모든 조합을 탐색하여 최적 파라미터를 찾고, 과적합 방지를 위한 워크포워드 분석을 지원한다.

## 2. 기능 요구사항

### FR-01: 그리드 서치 최적화
- 파라미터 범위를 지정하면 모든 조합을 백테스트
- 결과를 수익률/Sharpe/MDD 기준으로 정렬
- Top N 결과 반환

### FR-02: 워크포워드 분석 (Walk-Forward)
- 전체 기간을 in-sample / out-of-sample로 분할
- in-sample에서 최적 파라미터 탐색 → out-of-sample에서 검증
- 과적합 여부 판단

### FR-03: 결과 시각화
- 파라미터별 수익률 히트맵
- 최적 파라미터 조합 표시
