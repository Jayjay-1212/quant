# Plan: 전략 비교 분석 시스템

## 1. 개요

| 항목 | 내용 |
|------|------|
| Feature | strategy-comparison |
| 작성일 | 2026-03-06 |
| 상태 | Plan |
| 우선순위 | 2 |

### 1.1 목적
동일 종목/기간에 대해 여러 전략을 실행하고 성과를 비교 분석하는 시스템을 구축한다.

## 2. 기능 요구사항

### FR-01: 일괄 백테스트
- 전략 리스트를 받아 동일 조건으로 순차 실행
- 각 전략별 결과를 dict에 수집

### FR-02: 비교 테이블
- 수익률, CAGR, MDD, Sharpe, 승률을 한눈에 비교
- 랭킹 (수익률 기준 정렬)

### FR-03: 비교 시각화
- 전략별 equity curve 중첩 차트
- 전략별 수익률 바 차트
- 전략별 MDD 비교 차트
