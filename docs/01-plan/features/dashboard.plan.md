# Plan: Streamlit 웹 대시보드

## 1. 개요

| 항목 | 내용 |
|------|------|
| Feature | dashboard |
| 작성일 | 2026-03-06 |
| 상태 | Plan |
| 대상 파일 | dashboard.py |

### 1.1 목적
Streamlit 기반 웹 대시보드를 구축하여 종목 분석, 전략 백테스팅, 결과 비교를 브라우저에서 인터랙티브하게 수행할 수 있도록 한다.

## 2. 기능 요구사항

### FR-01: 종목 분석 페이지
- 종목 코드/기간 입력 → 주가 차트 + SMA 오버레이
- 기술적 지표 (RSI, MACD, 볼린저밴드) 표시

### FR-02: 백테스팅 페이지
- 전략 선택 + 파라미터 슬라이더
- 실시간 백테스트 실행 + 결과 표시
- Equity curve, drawdown 차트

### FR-03: 전략 비교 페이지
- 다중 전략 선택 → 동시 비교
- 성과 테이블 + 비교 차트

### FR-04: 포트폴리오 페이지
- 종목 리스트 + 비중 입력
- 리스크 패리티 자동 계산
- 포트폴리오 백테스트 결과
