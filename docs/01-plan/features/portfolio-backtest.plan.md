### 1.1 목적
여러 종목에 동일 전략을 적용하거나, 종목별 가중치를 지정한 포트폴리오 수준의 백테스팅을 지원한다.

## 2. 기능 요구사항

### FR-01: 다중 종목 데이터 로드
- 종목 리스트와 기간을 입력받아 일괄 데이터 로드

### FR-02: 동일 전략 다중 종목 적용
- 하나의 전략을 여러 종목에 각각 적용 후 개별 결과 산출

### FR-03: 포트폴리오 집계
- 종목별 가중치 기반 통합 수익률 계산
- 포트폴리오 전체 MDD, Sharpe 산출
